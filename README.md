# Title
  "Classifying Breast Ultrasound Images Using ML"
## Team Member(s)
  Kaitlin Simons (GitHubUserA)
## Project description
  The dataset I plan to use is Breast Ultrasound Images Dataset from Kaggle (cited source: Al-Dhabyani W, Gomaa M, Khaled H, Fahmy A. Dataset of breast ultrasound images. Data in Brief. 2020 Feb;28:104863. DOI: 10.1016/j.dib.2019.104863). It contains 780 breast ultrasound images categorized into normal, benign, and malignant as well as ground-truth masks for each image. The data was collected in 2018 from 600 female patients ages ranging 25-75 years old. Each ultrasound image is formatted as a png file and has an average image size of 500x500 pixels.
  The goal of the project is to develop a neural network capable of classifying breast ultrasound images into normal, benign, and malignant. The neural network will be trained using cross-entropy loss and the Adam optimizer which are good choices for multi-class image classification. I aim to demonstrate how computational tools can support early breast cancer detection by identifying patterns in ultrasound images that distinguish benign from malignant tissue.
